## Story

**As a** potential merchant,  
**I want** to see a compelling landing page that explains Orylo's value proposition and pricing,  
**so that** I can understand how Orylo protects my Stripe account and decide to sign up for a free trial.

## Dev Notes

### Relevant Architecture

**Positioning & Messaging** (from `docs/session-3-gtm-in-progress.md` and `docs/gtm-strategy.md`):

**Hero Headline:**
> "Ne laissez plus Stripe fermer votre compte à cause de fraudes.  
> Orylo détecte et bloque les attaques de card testing que Stripe Radar manque, avec une IA qui apprend des fraudes détectées chez 100+ marchands pour mieux vous protéger."

**3 Key Differentiators:**

1. **Protection Réelle** (vs Faux Sentiment de Sécurité)
   - Claim: Orylo = 95%+ détection card testing vs Stripe Radar 60-70%
   - Proof Point: "Sauvez votre business, pas juste vos transactions"

2. **Intelligence Collective** (Network Effect)
   - Claim: Fraudeur détecté chez Marchand A = protège B, C, D automatiquement
   - Proof Point: "Rejoignez 100+ marchands qui partagent la détection"

3. **Empathie & Contrôle** (Made for You)
   - Claim: Votre business est unique, votre protection aussi
   - Proof Point: "Personnalisez chaque règle, gérez chaque client"

**Pricing** (from `docs/gtm-strategy.md`):

- **Free Tier**: 0€/mois, jusqu'à 10K€ CA/mois
  - Détection basique (niveau 1 IA)
  - Blocage automatique fraudes évidentes
  - Dashboard basique
  - Support communauté

- **Standard Tier**: 99€/mois, jusqu'à 50K€ CA/mois (RECOMMENDED)
  - Détection standard (niveau 2 IA)
  - Règles custom (10 règles)
  - Dashboard complet
  - Support email

- **Pro Tier**: 399€/mois, illimité CA
  - Détection avancée (niveau 3 IA)
  - Règles custom illimitées
  - Dashboard avancé + API
  - Support prioritaire

### Design Patterns

**Hero Section:**
- Full-width background (subtle gradient or solid color)
- Centered content with max-width container
- Large headline (text-4xl md:text-5xl)
- Subheadline (text-lg md:text-xl, text-muted-foreground)
- Primary CTA: Large button, prominent color
- Secondary CTA: Outlined button or text link

**Features Section:**
- Grid layout: 1 column mobile, 3 columns desktop
- Card-based design with icons
- Title, description, proof point for each feature
- Icons: Shield (Protection), Network (Intelligence), Settings (Contrôle)

**Pricing Section:**
- Grid layout: 1 column mobile, 3 columns desktop
- Card for each tier
- Recommended tier highlighted (border accent color or badge)
- Feature list with checkmarks (Lucide Check icon)
- CTA button on each card
